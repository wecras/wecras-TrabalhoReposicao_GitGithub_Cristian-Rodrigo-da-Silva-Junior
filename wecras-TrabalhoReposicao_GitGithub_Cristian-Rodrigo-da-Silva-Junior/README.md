# Trabalho de Reposição — Git e GitHub
git init

Inicializa um repositório Git local.

git add

Adiciona arquivos à área de staging.

git commit -m "mensagem"

Registra uma versão dos arquivos no repositório.

git status

Mostra o estado atual dos arquivos monitorados.

git branch

Lista branches existentes / cria novas branches.

git checkout NOME_DA_BRANCH

Troca de branch.

git merge NOME_DA_BRANCH

Faz a fusão de uma branch na branch atual.

git push

Envia commits para o repositório remoto.

git pull

Atualiza sua branch local com alterações do remoto.

git log --oneline

Mostra o histórico de commits de forma resumida.
